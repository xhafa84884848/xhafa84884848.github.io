Webブラウザで離れた場所からPCをコントロール！

◆ホストPC（操作される側のPC）にインストールするだけで、ゲストPC（操作する側のPC）のJava対応（Sun Java 2）Webブラウザからリモートコントロールするソフトウェアです。

◆セットアップは簡単。ホストPC（操作される側）に「Desktop On-Call」をインストールするだけ。
ゲストPC（操作する側）にはインストールする必要がありません。
よって、ソフトウェアのライセンスはホストPCだけで低コストのリモートコントロール環境が構築できます。

◆「Desktop On-Call」は専用のモニタリングソフトを使用せず、ゲストPCのInternet ExplorerなどWebブラウザからWebページにアクセスする感覚で、リモートコントロールを行います。
操作する側のゲストPCを限定しないため、自宅のPCや出張先のホテルのPC、または携帯からホストPCへアクセスが可能です。

  • OS：Windows 11、Windows 10、Windows 8.1、Windows 8、Windows 7、Windows Vista、Windows XP、Windows 2000、 Windows Server 2012 R2 Windows Server 2012 Windows Server 2008 R2 Windows Server 2008、Windows Server 2003、Windows 2000 Server（各32-bit/64-bit）
    ※Windows 7 以上でご使用の場合は最新のアップデータを適用してください。
    ※英語版のインストーラーも含まれており、日本語以外のOSでもご使用になれます。

  • HDD：16MB以上の空き容量
  • 光学ドライブ：CD-ROMドライブ
  • ネットワーク：ネットワーク接続TCP/IP（グローバルIPアドレスが必要）
※ウィルス対策ソフトやファイアウォール機能をご利用の場合、設定を変更しないとご利用いただけない場合があります。
