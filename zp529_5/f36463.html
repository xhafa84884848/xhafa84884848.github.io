Ikko ITX01 ドッキングステーション USB-C ハブ - 10-in-1 USB C PD Dock DAC搭載 USB 3.2 ポート 10Gbps高速転送 4K/60Hz HDMI出力ポート microSD＆SDカードスロット 3.5mm&4.4mm オーディオ出力ポート Android、IOS、Mac OS、Win10、PS5、Switch等の Type C デバイス対応

高解像度の音楽データ - DACデコーダチップを搭載した最初のドッキングステーションとして、ITX01はiKKOが共同開発したSNC8600チップを使用し、Hi-Res小金標と音質品質を持ち、3.5 mmと4.4 mmのオーディオ出力をサポートし、タッチパネルで音楽、ゲーム、映画の3つのモードを切り替えることができ、異なるシーンにより詳細な音声表現を提供することができます。( ITX01に搭載されたDACチップはハイパワーで、雑音を避けるために、32[オーム]以上のイヤホンを使用するの方がおすすめです。)(サウンドモードの切り替えるのは、タッチパネルを3秒以上で長押しする必要です。 ちなみに、押すときに、タッチパネル全体を完全に被さてください。)
圧倒的な転送速度 - iKKO タイプc ドッキングステーションがUSB 3.2プロトコルをサポートし、10 Gbps超高転送速度で秒転送を実現し、Huawei / Samsung-デスクトップモードをサポートし、携帯を数秒でコンピューターに変更できます。
豊富な接続ポート - ハブとして10つのポートをひとつに:10Gpbs伝送速度 USB3.2ポート3、4K60Hz出力HDMIポート、100W PD Type Cポート、データ Type Cポート、SDカードスロット、TFカードスロット、3.5mm& 4.4mmオーディオポートを備え、あらゆる拡張性を実現します。
安心的な熱暴走対策 - iKKO ITX01 usb type-c ハブは、陽極酸化技術による航空アルミニウム合金ボディで作られています。これにより、機械の耐摩耗性、耐食性、品質の向上、指紋が付着しにくく、放熱が速く、手触りが快適で、商品内部の発熱量を製品表面に導きやすく、放熱を加速させています。
幅広い互換性 - ITX01ドッキングステーション はM1チップを搭載したMacBookにも対応でき、ポートが少ない不便とは無縁となり、そして、Android携帯とApple携帯の転送画面表示をサポートしています。USBポートハブ及びType Cハブを搭載し、PCでもSwitchなどのゲーム機でも、即、仕事/娯楽と繋がれます。
