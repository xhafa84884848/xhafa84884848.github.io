この Raspberry Pi Mini Tower NAS キットには、Raspberry Pi の大容量ストレージを拡張できる M.2 SATA SSD 拡張ボードが含まれています。ケースはABS素材でできており、このキットはRaspberry Piをモバイルミニホストに変えます。
ケースにはすでに0.96インチのOLEDスクリーンが搭載されていました。 Raspberry Pi の IP アドレス、CPU 温度、ディスク使用率、CUP 負荷、メモリ ステータス、その他のシステム情報など、プログラミングを通じて Raspberry Pi の実行状態を表示できます。独自のカスタマイズによって表示内容を変更する
注: ファンとムード ライトの RGB ライトは、ドライバーがインストールされていない限り点灯しません。OLED は、有効にするまで情報を表示しません。マニュアルをよく読み、手順に従って有効にしてください。M.2 SATA SSD 2280 大容量記憶装置を購入してください。NVME SSD には対応していません!!!
M.2 SATA SSD 2280/2260/2240 大容量ストレージ アダプター ボードは、最大 2TB の M2 をサポートします。 SATA SSD ソリッドステート ドライブ
パッケージ内容: 1 x ABS ミニタワーケース、1 x QC3.0 電源、1 x ICE-Tower CPU 冷却ファン、2 x アクリルサイドパネル、1 x 0.96 インチ OLED ディスプレイ (ケースに取り付け済み)、1 x GPIOエッジ拡張ボード、1 x M.2 SATA SSD シールド、1 x 組み立てマニュアル (注: Raspberry Pi 4B はパッケージに含まれていません)
