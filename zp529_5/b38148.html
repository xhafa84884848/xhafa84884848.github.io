最高品質の配信


1080p 30fps のフル HD 配信を実現する USB Web カメラ、Razer Kiyo X で最高品質の配信を実現。
優れた映像品質、完全にカスタマイズ可能な設定、スマートなオートフォーカスなど、最高のコンテンツを作るためのすべての機能を備えています。

非常に鮮明で滑らか

高精細映像を求めるユーザーであれ、より滑らかな配信を目指すユーザーであれ、Razer の USB Web カメラならどちらの要望にも応えることができます。

完全にカスタマイズ可能な設定

お気に入りの設定を残しておいたり、いくつかの設定を切り替えたければ、Razer Synapse でビデオ設定を保存しておいて、カスタムプロファイルを簡単に切り替えることで、どのような状況でも完璧な映像を映すことができます。

  • ・オートフォーカスとマニュアルフォーカスを切り替え
  • ・さまざまな画像プリセットを選択
  • ・輝度、コントラスト、彩度、ホワイトバランスの調節
  • ・カメラ設定のプリセットを選択して素早く配信開始

柔軟な取付けオプション

モニターにぴったりフィットする安定した Web カメラマウントで常に完璧な角度を維持でき、上下に傾けたり、取り外して Razer Kiyo X を三脚に取り付けることができます。

プラグアンドプレイ

すぐに配信を開始したければ、Razer Kiyo X は、Razer Synapse 3 により箱から取り出してすぐに使用でき、一般的な配信プログラムに対応しているため、最小限の操作でセットアップ環境にシームレスに追加できます。
