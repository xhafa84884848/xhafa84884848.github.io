Plugable USB 2.0 デジタル顕微鏡 マイクロスコープ（200万画素 10倍～250倍ズーム）- 送料無料


デジタル・マイクロスコープ（顕微鏡）で広がる楽しみ

この Plugable USB 2.0 デジタルマイクロスコープ（USB2-MICRO-250X）は、自由研究に活用したり、手軽なツールとして研究や趣味に、または企業で業務上必要な場合に、簡易顕微鏡としてお使いいただくのに適しています。


理科実験や観察、切手やコインの状態チェック、極小部位のはんだ付け、ガーデニングの際の害虫チェックなど、その利用範囲は多岐に渡ります。


USB バスパワーのため、電源はコンピュータの USB ポートから供給されます。このマイクロスコープ用の電源アダプタなどを使用する必要はありません。


当デジタル・マイクロスコープ用の無料ソフトウェア Digital Viewer をダウンロードしてお使いいただくことができます。Digital Viewer ソフトウェアは Windows および macOS 対応で、導入すると簡単に当製品のカメラ機能等を利用できます。また、クリックで画像キャプチャの解像度や露出も簡単に変更でき、動画取得機能や連続キャプチャ機能もありますので、被写体の連続観測をするのにも役立ちます。


Digital Viewer ではなく、他のウェブカメラ対応のソフトウェアをご利用いただくことも可能です。










LED ライト内蔵

マイクロスコープのレンズのそばには LED ライトが組み込まれており、輝度レベルをダイヤル調整することで、適切に被写体を照らすことができるようになっています。他の多くの USB デジタル顕微鏡の LED とは異なり、この LED ライトは反射しすぎないように光拡散機能を備えています。


この光拡散機能はコインのような反射物体を観察する場合には特に便利です。この拡散気が均等に光を照射することで、対象物の表示の際に光を反射しすぎないようになっています。


この LED ライトは、マイクロスコープを PC の USB ポートに差し込み、ソフトウェアでこの顕微鏡を使用デバイスとして指定すると、自動的に点灯します。


画像キャプチャ用ボタン

画像キャプチャ機能を提供するボタンは、物理的なプッシュボタンではなく静電容量式タッチボタンです。物理ボタンに比べてアームを揺らさずに済むため、高倍率で表示しているような際にもブレを最小限に抑えます。


画像キャプチャや動画取得機能、連続キャプチャ機能などは、ウェブカム用のソフトウェアや Digital Viewer を介して利用できます。


多目的スタンド

柔軟に曲がるネック部分を好きな形に変えて、大小の被写体の表面を任意の角度で簡単に分析することができます。アーム下部の吸盤カップを目盛り付き観察用プレートの滑らかな面に取り付けて使用すると、目盛りで標本を簡易測定することができます。



互換性

Windows


ウェブカメラ用（USB Video Class、UVC）のドライバは Windows 11、10、8.ｘ に内蔵されています。 任意で使用できる無料ソフトウェア Digital Viewer がダウンロードできます。また、それ以外のウェブカメラ対応のソフトウェアからも、外部カメラとしてご利用いただけます。


macOS と OS X


Mac OS X 10.4.2 以上であれば、このデジタル顕微鏡に必要な USB ビデオクラス・ドライバが内蔵されています。 任意で使用できる無料ソフトウェア Digital Viewer がダウンロードできます。また、Photo Booth 等のビデオ・キャプチャ用ソフトウェアでもご利用いただけます。


Linux


Linux カーネル 2.6.26 以上であれば標準でサポートされます。 Raspberry Pi Os で動作します。Cheese または GTK+ UVC Viewer の使用をお勧めします。


ChromeOS および Android


ChromeOS 機器（ChromeBook）は OS 内蔵のカメラアプリケーションにより使用できます。Android 機器に関しては、このマイクロスコープが使用できるかどうかはその機器に依存しています。新しい製品の多くは OTG ケーブルとウェブカム用アプリケーションを利用すればこのマイクロスコープが機能しますが、一部の Android 機器では OTG や必要な USB クラスに非対応で、使用できないことがあります。外部カメラをサポートするかどうかは、Andorid デバイスメーカー次第です。正式な情報は、お使いの Android デバイス・メーカーに確認してください。


iOS


iOS デバイス（iPhone、iPad）は外部カメラに対応していないため、当マイクロスコープ を使用することはできません。










倍率

このマイクロスコープは、まず 40 倍の倍率でオブジェクトを表示します。解像度と画像を見ながら、表示倍率を調整してください。


最大 250 倍表示

250 倍に調整すると、人間の目には見えない細胞や粒子などミクロの世界を見ることができます。
