ラズベリーパイ コンピュートモジュール4用 ミニベースボード CM4-IO-BASE-B Mini Base Board for Raspberry Pi Compute Module 4

Raspberry Pi CM4の評価や最終製品への組み込みに適した、Raspberry Pi Compute Module 4用に設計されたベースボードです。

標準的なRaspberry Pi CM4ソケットと色分けされたRaspberry Pi 40ピンgpioヘッダは、Compute Module 4 Lite/EMMCシリーズモジュールに適しています。

複数のコネクタを搭載： CSI/DSI/RTC/FAN/HDMI/USB/RJ45ギガビットイーサネット/マイクロSDカードスロット/M.2スロット。DSIディスプレイ・インターフェースはDSI0インターフェースで、DSI1ディスプレイ・インターフェースは接続されていません。

FANファンは5Vファンのみサポート、12Vはサポートしません、接続前にファン電圧を確認してください。外形寸法：85x56mm。

取り付けネジ付き、取り付け簡単。M.2インターフェイスを使用する場合、適合するネジを使用してください。
