Club 3D HDMI オス to USB Type C メス 4K120Hz 8K30Hz アクティブ アダプタ (CAC-1336)

Club 3D CAC-1336 は、8K30Hz または 4K120Hz解像度をサポートするHDMIソースから、USB Type C 8K30Hz または 4K120Hz対応ディスプレイへ接続するための最適なソリューションです。
入力コネクタは1x HDMI (オス)・1x Micro USB (メス)、出力コネクタは1x USB Type C (メス)です。ケーブルの長さは約26cm、商品の重さは28g、ボックス込みの総重量は99gです。
USB DP Alt Mode 仕様（最大3840x2160/120Hz または 7680x4320/30Hz）および HDMI 仕様（最大3840x2160/120Hz または 7680x4320/30Hz）に準拠します。サポートする解像度/リフレッシュレートは、最大4K120Hz または 8K30Hzです。アアダプタは、Micro USBメスポートから別途電源供給が必要です。給電には、付属のUSB Type A to Micro USB M/M 1mケーブルを使用します。
対応OSは、Microsoft Windows および Mac OSとなります。ROHS（特定有害物質の使用制限）、FCCとCEマークのEMI要求（人体や他の機器への電磁妨害）に適合しています。「国内正規保証」。保証期間は購入から１年となります。【注意】VRR（可変リフレッシュレート）はグラフィックドライバがサポートしてないため動作致しません。
