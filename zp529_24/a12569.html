この商品について
DisplayPort映像出力をもつゲーミングPCやラップトップPCおよび映像機器とデュアルリンク DVI-D映像入力をもつモニター、プロジェクタに接続するのに最適な「HDCPに対応」したアダプタです。
入力側はDispalyPortオスコネクタおよび電源供給用のUSB Type Aオスコネクタ、出力側は、デュアルリンク DVI-D メスコネクタです。アダプターのサイズは、1.4 x 6.1 x 8.3 cm、重さは109gです。アダプタ全体の長さは約60cmです。DisplayPort端子は、バージョン1.2およびAdaptive-Syncに対応したバージョン1.2aをサポートしています。
サポート解像度は、30インチApple Cinemaディスプレイにおいて2560 x 1600@50Hzです。デュアルリンク DVI-D 最高映像解像度では、WQXGA 2560 x 1600 @ 50Hzです。デュアルリンク DVI-D 最高映像解像度では、 3840*2160 @ 30Hzです。シングルリンク DVI 最高映像解像度では、 1920 x 1080となります。アダプターへの給電はDisplayPort端子とUSB Type A端子から行います。アダプタのモニタ側端子のデュアルリンク DVI-Dは、2560 1600解像度のデュアルリンクおよび1920 1200解像度のシングルリンクに対応しています。
CAC-1010は、HDCPをサポートするディスプレイ向けのHDCPバージョンのアダプタです。PCなどのソースデバイスやディスプレイがHDCPをサポートしていない場合は、HDCP OFFバージョンの CAC-1010-A をご購入ください。
Microsoft Windows、Apple OS X、Google Chrome OSに対応しています。ROHS 特定有害物質の使用制限 、FCCとCEマークのEMI要求 人体や他の機器への電磁妨害 に適合しています。「国内正規保証」。保証期間は購入から１年となります。
