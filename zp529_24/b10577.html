🚀【11-in-1 USB C ドッキング ステーション】複数のディスプレイ用のドック ステーション 2 HDMI、デュアル 4K HDMI、1 DP ディスプレイ、2 USB 3.0 ポート、USB C データ転送ポート (最大 5Gbps の速度)、USB C 電源配信ポート (最大 100 W)、ギガビット イーサネット、SD/Micro SD カード リーダー、および 3.5 mm マイク/オーディオ。
🚀【USBCハブはトリプル】HDMIおよびDPポートは最大4k @ 60Hzの解像度をサポートできます。 「コピー」モードでは 4 つの等しい画面が得られ、「拡張」モードでは 3 つの拡張ディスプレイが得られます. システムの制限により、Mac OS MST モードでは 1 つの画面しか拡張できません (つまり、3 つの画面に同じコンテンツが表示されます).
🚀【多機能 USB Cドッキングステーション】USB Type C ハブ 2 hdmi は、1*USB C、2*USB 3.0 ポート、および 1 つの SD/TF カード スロットを備え、転送速度は最大 5Gbps です.さらに、PD 3.0 電源がありますアダプタ インターフェイスを使用すると、出力電力が 100W 未満になる場合がありますが、これは通常の現象です; ギガビット イーサネット ポートも装備されており、安定した高速イーサネット ネットワーク状態を提供します。
🚀【4K超高解像度ディスプレイ】4K超高解像度の視覚的な饗宴を楽しむことができます。ディスプレイ解像度は最大4K @ 60Hzで、最適な解像度はコンピューターシステムによって異なります。 2 台のモニターを接続すると、1*4K@60Hz および 1*4K@30Hz ディスプレイが得られます。 3 台のディスプレイが同時に接続されている場合、1*4K/60Hz、2*4K/30Hz または 1*4K/60Hz、1*4K/30Hz、1*2K/60Hz が得られます。
🚀【優れた互換性】優れた互換性を備えた USB c ドッキング ステーション デュアル モニターは、市場に出回っているほとんどの Type-C ラップトップで動作し、MacBook、ChromeBook Pixel、Surface、HP、Lenovo などのラップトップと互換性があります。型式は詳細ページの説明欄をご参照ください 不明な場合はお気軽にお問い合わせください。）
