Raspberry Pi コンピュータモジュール 4 CM4 ルータキャリアボード

概要

  • Raspberry Pi CM4 デュアルギガビットルータボード
  • 統合型 0.91インチ OLEDディスプレイ および PoE HATに対応
  • Raspberry Pi CM4を強化
  • デュアルフルスピードギガビットポート搭載
  • 多用途のソフトウェア互換性
  • PWMファンにより冷却

Raspberry Pi コンピュータモジュール4 CM4 ルータキャリアボードは、Raspberry Pi Compute Module 4の機能拡張用の拡張ボードです。2個のフルスピードギガビットネットワークポートを搭載し、パフォーマンスの向上、CPU使用率の低減、長時間動作時の安定性の向上を実現します。このボードには従来の GPIOピンヘッダを維持しており様々なHATボード、センサ、冷却ファンを接続することができます。さらに、フラッシュドライブ、キーボード、プリンタ、Wi-Fiモジュールなどの周辺機器用の USB2.0インタフェースも搭載しています。

0.91インチの統合型OLEDスクリーンにより、情報表示をカスタマイズでき、ボードの機能に使いやすいタッチが加わりました。このボードは OpenWRTオープンソースルータシステムに対応し、簡単なセットアップ操作でルータとして機能することができます。また、Raspberry OS、Ubuntu Server、その他の Raspberry Piシステムとも互換性があり、レトロゲームを含む様々な用途に対応できます。

標準的な HDMIインタフェースを搭載していますので設定や操作情報の表示も簡単に行えます。BCM2711 Quad-Core 1.5GHz Cortex-A72プロセッサの強力な性能と、Raspberry Piコミュニティによる広範なソフトウェアサポートにより、このボードは高性能ゲートウェイ、スマートルータ、ホームIoTプラットフォームの構築に優れた基盤となっています。また、周辺機器と接続することで、ターミナル経由で制御可能なミニNAS、ワイヤレスネットワークブリッジ、ミニサーバとして使用することもできます。

複数のLANネットワークポートが必要な場合は、インターネットパフォーマンスを損なうことなくネットワークポートの拡張が可能なギガビットスイッチの使用をお勧めします。ボードの DC5Vファンインタフェースは冷却ファンに対応し、PWMプロトコルに対応しているため、PWM信号でファンの回転数を制御することができます。このボードは、ルータとして、または他のアプリケーションに使用するのに十分な汎用性がありますが、Raspberry Piボードは付属していませんのでご注意ください。

このボードを使用するには Raspberry Pi Compute Module 4 シリーズが必要ですが、別売となっていますので注意してください。
